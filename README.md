# Pong

Testing out how well Claude works in the CLI — built this pong game in a single session.

## Play the game

👉 **[Click here to play](https://kcdavs.github.io/Pong-Project/)**

## Controls

| Key | Action |
|---|---|
| `Arrow Up` | Move paddle up |
| `Arrow Down` | Move paddle down |
| `Enter` | Start / restart |

First to 7 points wins.
